Visión General del Sistema

Sistema SaaS multi-tenant que permite a múltiples empresas gestionar de forma independiente sus productos, ventas, proveedores, empleados y métricas financieras en un entorno seguro y aislado.

🏢

Multi-Tenancy

Aislamiento completo de datos entre clientes con arquitectura segura a nivel de aplicación.

📊

Gestión Integral

Control completo de productos, ventas, proveedores, empleados y métricas financieras.

🔒

Seguridad Avanzada

Autenticación JWT, autorización basada en roles y aislamiento de datos por tenant.

⚡

Alto Rendimiento

Arquitectura optimizada con Spring Boot 3, React 18 y infraestructura AWS escalable.

Ventajas de la Arquitectura Unificada

🔄

Desarrollo Sincronizado

Frontend y backend evolucionan juntos, eliminando incompatibilidades entre versiones.

🚀

Despliegue Atómico

Un solo commit puede actualizar toda la aplicación, garantizando consistencia total.

🛠️

Gestión Simplificada

Características Clave de Seguridad

🔐

Autenticación JWT

Tokens seguros con claims de usuario, rol y clientId. Validación automática en cada petición.

🏠

Multi-Tenancy

Aislamiento de datos a nivel de aplicación con filtrado automático por clientId en todas las consultas.

🛡️

Rutas Protegidas

Componente ProtectedRoute que valida autenticación antes de permitir acceso a secciones privadas.

Infraestructura AWS

🌐

Frontend (S3 + CloudFront)

Hosting estático en S3 con distribución global via CloudFront CDN. Configuración OAC para acceso seguro.

🖥️

Backend (EC2 + ECR)

Instancia EC2 con Docker ejecutando contenedores desde imágenes almacenadas en Amazon ECR.

🗄️

Base de Datos (RDS)

MySQL 8.0 en Amazon RDS con alta disponibilidad, backups automáticos y parches gestionados.

🔒

Seguridad (VPC + IAM)

VPC con subredes públicas/privadas, Security Groups restrictivos y roles IAM sin credenciales estáticas.

Pipeline CI/CD

🔍 Quality Check

Ejecuta en cada push. Compila el código Java y ejecuta pruebas unitarias con Maven para garantizar la calidad del código.

🚀 Frontend Deploy

Build con Vite, sincronización con S3 e invalidación de caché en CloudFront para servir la nueva versión.

🐳 Backend Deploy

Construcción de imagen Docker, push a ECR y despliegue automatizado en EC2 con rolling update.
